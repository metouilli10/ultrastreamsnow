```json
  "priceCurrency": "USD",
  "availability": "InStock"
}
```
**Purpose:** Helps search engines understand this is a free offer

```json
{
  "@type": "FAQPage",
  "mainEntity": [8 questions with answers]
}
```
**Purpose:** Enables rich snippets in search results with expandable Q&A

#### 3. Breadcrumb Schema

**Purpose:** Improves site structure understanding and SERP appearance
